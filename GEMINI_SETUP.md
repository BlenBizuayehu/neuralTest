## Troubleshooting

### "API key not configured"
- Make sure you selected "Gemini" before entering the key
- Check the key starts with `AIza`

### "Quota exceeded"
- You've hit the 15/minute limit
- Wait 1 minute and try again
- Or use OpenAI if you have credits

### Key doesn't work
- Make sure you copied the full key
- Try creating a new key at https://makersuite.google.com/app/apikey
- Check you're using the correct key (not OpenAI key)
